# ScanBill
QR-Based Self Billing and Payment System
QRBazaar is a QR-based self-billing and payment application for retail stores.
It allows store owners to add products, generate QR codes for each product, and export them as PDF.
Customers can scan product QR codes, add items to a cart, generate the final bill, and pay using UPI inside the app.
The system helps reduce billing time, improve customer convenience, and make store management easier.
It also supports owner account management and monthly subscription plans for store owners.

Short version
